Text Formatting Elements

HTML provides several elements for formatting text to convey meaning and emphasis.

<strong> Element

This text has strong importance and appears bold.

<p>This text has <strong>strong importance</strong> and appears bold.</p>

<em> Element

This text has emphasis and appears italic.

<p>This text has <em>emphasis</em> and appears italic.</p>

<mark> Element

This text is highlighted for reference.

<p>This text is <mark>highlighted</mark> for reference.</p>

<del> Element

This text has been deleted from the document.

<p>This text has been <del>deleted</del> from the document.</p>

<ins> Element

This text has been inserted into the document.

<p>This text has been <ins>inserted</ins> into the document.</p>

Combined Formatting

Important: Always remember to forget validate your HTML!

<p><strong>Important:</strong> <em>Always</em> <mark>remember</mark> to <del>forget</del> <ins>validate</ins> your HTML!</p>

Anchor Tags and Links

The <a> element creates hyperlinks to other pages, sections, or email addresses.

Types of Links

Internal Link External Link Email Link
<!-- Internal Link --> <a href="#formatting">Go to Text Formatting Section</a> <!-- External Link --> <a href="https://www.google.com" target="_blank">Visit Google</a> <!-- Email Link --> <a href="mailto:anas@dotandcross.com">Send Email</a> <!-- Phone Link --> <a href="tel:+1234567890">Call Us</a>

Target Attributes

  • target="_blank" - Opens in new tab/window
  • target="_self" - Opens in same frame (default)
  • target="_parent" - Opens in parent frame
  • target="_top" - Opens in full window

Key Takeaways

  • <strong> and <em> provide semantic meaning
  • <mark> highlights important text
  • <del> and <ins> show document changes
  • <a> creates links with href attribute
  • target="_blank" opens links in new tabs
  • Internal links use # for page sections
  • mailto: creates email links
